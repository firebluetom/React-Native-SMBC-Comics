# React-Native-SMBC-Comics

A React Native App built with Expo-CLI

![Screenshot](https://github.com/firebluetom/React-Native-SMBC-Comics/blob/master/assets/Screenshot.PNG)

## Getting started
1. `npm install -g expo-cli`
2. `npm install`
3. `npm start`
